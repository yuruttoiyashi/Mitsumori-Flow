# Mitsumori Flow

物流会社向けの見積管理システムです。  
見積作成、保存、一覧管理、プレビュー、印刷、CSV一括取込までを一連の業務フローとして扱える、実務感を意識したポートフォリオアプリです。
- Demo: https://mitsumori-flow-6s79.vercel.app
- GitHub: https://github.com/yuruttoiyashi/Mitsumori-Flow
---

## 概要

Mitsumori Flow は、物流業務における見積作成・管理を効率化するための Web アプリケーションです。  
「案件ごとの見積をすばやく作成し、保存し、一覧で管理し、見積書として印刷できる」流れを、企業向けUIでまとめています。

単なるフォームアプリではなく、以下のような実務で想定される業務導線を意識して設計しています。

- 新規見積の作成
- 既存見積の編集・削除
- 一覧での確認
- ダッシュボードでの集計確認
- 見積書プレビュー
- 印刷出力
- CSV によるデモデータ一括投入

---

## 開発背景

物流現場では、荷主企業ごと・案件ごとに見積作成が発生し、過去見積の参照や修正、印刷提出などの業務が日常的に発生します。  
本アプリでは、そうした流れをシンプルなUIで一元管理できるようにし、「現場で使う業務システムらしさ」を重視して開発しました。

また、会社提出用ポートフォリオとして、以下の点が伝わるよう構成しています。

- 実務を意識した画面構成
- 業務に沿った文言設計
- データ保存を伴うCRUD実装
- 印刷やCSV取込など、現場運用を意識した機能

## 主な機能

### 1. 物流見積の新規作成
- 荷主企業
- 荷主担当者
- 案件名
- 発行日
- 有効期限
- 見積明細
- 備考

を入力して、見積を登録できます。

### 2. Firestore への保存
作成した見積データは Firebase Firestore の `quotes` コレクションへ保存されます。

### 3. 物流見積一覧
登録済みの見積を一覧で確認できます。  
案件名、荷主企業、金額、作成日などを一覧で把握できる構成です。

### 4. 見積編集・削除
既存の見積データを編集・削除できます。

### 5. ダッシュボード表示
以下のような業務指標を可視化しています。

- 総案件見積数
- 今月登録件数
- 運送見積総額
- 最近の物流見積

### 6. 見積書プレビュー
印刷前に見積書レイアウトを確認できます。  
会社提出向けを意識し、業務帳票らしい見た目に寄せています。

### 7. 印刷対応
プレビュー画面から印刷できるようにしています。

### 8. CSV一括取込
テンプレートCSVを使って、デモデータを一括投入できます。  
ポートフォリオ閲覧時にも、一覧やダッシュボードにデータが入った状態をすぐ再現できます。

## 実装上のこだわり

### 業務システムらしい文言設計
汎用的な「会社名」「顧客名」ではなく、物流業務に寄せた表現へ統一しています。

例:
- 顧客名 → 荷主担当者
- 会社名 → 荷主企業
- 件名 → 案件名
- 見積一覧 → 物流見積一覧
- 見積総額 → 運送見積総額

### 見積明細の入力しやすさ
見積明細の「内容」欄は、入力しやすさを重視して横幅を大きく確保し、  
その下に単価・数量・金額を並べることで、実務で使いやすい構成にしています。

### 提出向けの視認性
- 左サイドバー上部にロゴを配置
- アプリ名は主張しすぎず、業務アプリらしい落ち着いたUI
- プレビュー・印刷画面も帳票として見やすい構成

### デモしやすさ
CSV一括取込により、企業提出時や面談時にも短時間でデータ入りの状態を再現できます。
